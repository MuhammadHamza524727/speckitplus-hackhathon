# Test RagChatbot Component

This page is designed to test the functionality of the RagChatbot component.

## How to Test

1. Select any text on this page
2. An "Ask AI" button should appear near your selection
3. Click the button to send the selected text to the `/rag-query` endpoint
4. The AI response should appear in a modal
